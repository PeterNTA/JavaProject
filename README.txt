Trên đây là link chưa source code của toàn bộ đồ án.
Có file csdl mẫu tên “new.sql”.
Đồ án được sử dụng NetBeans làm công cụ phát triển chính. 
Dùng Java Platform chính là JDK 15 (default).
Hiện tại đồ án này chưa có khả năng cài đặt cụ thể lên một máy khách, chỉ có thể build trực tiếp trên NetBeans và sử dụng XAMPP MySQL localhost để kết nối cơ sở dữ liệu.
Dùng các lệnh query trong file “new.sql” để tạo csdl local.
